# ServiConnect

Plataforma web para conexão entre clientes e prestadores de serviços.

## Tecnologias
- Flask
- MySQL
- HTML/CSS/JS

## Funcionalidades
- Login
- Cadastro
- OAuth Google
- Perfil
- Marketplace
